# FinanAcess – Tela Inicial (FIAP - Prototyping / Desenvolvimento Web)

Criação de uma tela do protótipo desenvolvido na Fase 2, utilizando HTML, CSS e Bootstrap, seguindo as orientações do exercício.

---

## Objetivo do Projeto
Criar uma única tela do aplicativo FinanAcess, mantendo:
- Identidade visual do protótipo criado anteriormente
- HTML e CSS separados
- Layout responsivo
- Uso de Bootstrap
- Organização de pastas e arquivos
- Imagens em diretório próprio

---

## Tela Desenvolvida
A tela implementada corresponde à **tela inicial de boas-vindas**, contendo:

- Ilustração principal  
- Título e subtítulo  
- Indicadores de navegação  
- Botão “Criar uma conta”  
- Link “Fazer login” com efeito hover  
- Layout centralizado  
- Comportamento totalmente responsivo
